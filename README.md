# Utility Front End Stack
A modern frontend stack built with Next.js for mapping and utility visualization, designed to connect 
seamlessly with a Django backend hosted on a separate VPS.

![Map Image](/public/image/image.png)

---
Features
- Interactive maps with ESRI icons for points of interest
- Demo data visualized using GeoJSON
- Fully decoupled frontend and backend for easy maintenance
- Ready for production deployment on Vercel
- Modular and scalable Next.js project structure

---
TechStack
- frontend: nextjs
- mapping library: maplibre
- hosting: vercel(optional)
- backend django(separate vps)
- mapping icons: Esri Calcite
- data: geojson


---
Get Started
```
npm install
npm run dev
```
